# 🔐 Phishing Awareness Training Module

## 📌 Description
The **Phishing Awareness Training Module** is a Streamlit-based web application designed to educate users about phishing attacks and online security. It helps users identify fake emails, understand social engineering tactics, and learn best practices to stay safe online.

---

## 🚀 Features
- 📧 Recognize phishing emails and fake websites  
- 🧠 Learn common social engineering techniques  
- ✅ Best practices for cybersecurity awareness  
- 🌍 Real-world phishing examples  
- 📝 Interactive quiz to test knowledge  

---

## 🛠️ Tech Stack
- Python 🐍  
- Streamlit 🌐
